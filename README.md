
# FaithQuest — Family Bible Learning

FaithQuest is a family-first Bible learning concept:
- Short Scripture
- Age-tailored understanding
- Guided family discussion
- Offline “faith-in-action” missions
- Reflect & pray together

This repo currently contains the **marketing/landing experience** (static HTML).
